### Project setup
Feel free to use `NPM` or `YARN`
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

Deploy to github pages
```
sh deploy.sh
```

### Test

Unit Testing:
```
yarn test:unit (--coverage)
```

E2E Testing:
```
yarn test:e2e
```
